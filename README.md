# Music Store Power BI Dashboard

## Project Overview
This project focuses on analyzing music store sales data using Power BI to identify customer trends, employee performance, revenue distribution, and business insights.

## Dashboard Preview
Dashboard screenshot and walkthrough video are available in this repository.

## Tools Used
- Power BI
- SQL
- Excel

## Key Business Metrics
- Total Customers: 60
- Total Invoices: 413
- Average Sales: 43.94
- Total Revenue: 3329

## Business Insights
- Jane generated the highest sales among employees.
- Revenue peaked in 2011.
- Top customers contributed significantly to total revenue.
- Country-level sales patterns were identified through geographic analysis.
